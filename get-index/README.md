## `getIndex`

### Usage

```javascript
getIndex(document.querySelector('.list-item.active'))
```

Will return `2`.

### Example

```html
<ul class="list">
    <li class="list-item"></li>
    <li class="list-item"></li>
    <li class="list-item active"></li>
    <li class="list-item"></li>
    <li class="list-item"></li>
</ul>
```

### Parameters

#### `first parameter`

`HTMLElement`

Tells to the function the target element.
